# Eras Tour: Warehouse Insights

A data engineering pipeline and dashboard designed to analyze warehouse logistics, track inventory, and identify operational bottlenecks for high-scale events.

## 💻 Tech Stack
*   **Languages:** Python, SQL
*   **Frameworks:** Streamlit, FastAPI

## ⚙️ Quick Start

1. **Clone the repo:**
   ```bash
   git clone [https://github.com/thethmue04/Eras-Tour-Warehouse-Insights.git](https://github.com/thethmue04/Eras-Tour-Warehouse-Insights.git)
   cd Eras-Tour-Warehouse-Insights

2. **Set up the environment :**
   ```bash
   python -m venv .venv
   .venv\Scripts\activate
   pip install -r requirements.txt

3. **Run the app**
   ```bash 
   streamlit run dashboard.py
